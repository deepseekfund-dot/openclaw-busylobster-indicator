# OpenClaw BusyLobster Indicator 🦞

A UI enhancement plugin for OpenClaw. Displays a cute running lobster animation after 1.5 seconds to soothe user waiting anxiety during AI task processing.

## Features

- **Golden Delay Trigger**: Only appears after 1.5s to avoid disrupting short tasks
- **Elegant Fade Effects**: 0.4s smooth fade in/out transitions
- **CDN Accelerated**: Powered by jsDelivr for global fast access
- **Zero Overhead**: Auto preloads and cleans up

## Installation

```bash
clawhub install openclaw-busylobster-indicator
```

## Usage

This plugin automatically shows the busy indicator when OpenClaw is processing tasks.
