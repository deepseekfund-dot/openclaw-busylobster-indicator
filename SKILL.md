# OpenClaw BusyLobster Indicator 🦞⚡

A UI enhancement plugin for OpenClaw. Displays a cute running lobster animation with lightning effect after 1.5 seconds to soothe user waiting anxiety during AI task processing.

## Features

- **Native Emoji**: Uses 🦞 (Lobster) + ⚡ (Lightning) with pure CSS animations
- **No External Assets**: No GIF files needed, works everywhere
- **Golden Delay Trigger**: Only appears after 1.5s to avoid disrupting short tasks
- **Elegant Animations**:
  - Lobster running/hopping animation
  - Lightning pulse effect
  - Dust cloud particles
  - Smooth fade in/out transitions
- **Dark Mode Support**: Auto-adapts to system theme
- **Zero Overhead**: Auto preloads and cleans up

## Installation

```bash
clawhub install openclaw-busylobster-indicator
```

## Usage

This plugin automatically shows the busy indicator when OpenClaw is processing tasks.
